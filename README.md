# FOCUSVERSE

A cinematic scroll-powered landing page for a Pomodoro planner and productivity app.

## Stack

- Next.js App Router
- React and TypeScript
- Tailwind CSS
- Framer Motion
- GSAP ScrollTrigger
- Lenis smooth scrolling
- Three.js, React Three Fiber, and Drei
- Recharts

## Run Locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Production Check

```bash
npm run typecheck
npm run build
```

Editable landing-page content lives in `lib/site-data.ts`.
